# hub-utility-leyu

本仓库用于归档和发布多个独立的 HTML 页面，内容涵盖各类实用工具、信息展示或功能演示页面。所有页面均为独立的静态文件，不依赖特定域名或网站。

## 目录说明

```
/
├── index.html            # 仓库首页 / 导航页
├── pages/                # 存放各独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md             # 本文件
```

- `index.html`：仓库的入口页面，通常包含页面列表或导航链接。
- `pages/`：所有独立 HTML 页面均放置在此目录下，便于管理和引用。

## 页面归档说明

- 每个页面均为完整的 HTML 文件，可直接在浏览器中打开。
- 页面之间无依赖关系，可独立使用。
- 页面内容可能包含工具、演示、信息展示等，具体功能请参阅各页面内说明。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 以改进现有页面或添加新页面。
- 新增页面请放入 `pages/` 目录，并在 `index.html` 中添加对应链接。
- 保持页面结构清晰、代码整洁，避免引入外部不可控资源。
- 本仓库仅用于归档和发布，不保证页面内容的持续可用性或更新频率。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等服务进行在线访问（需自行配置）。
